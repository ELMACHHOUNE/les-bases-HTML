Layout & Semantics

Use semantic tags to describe page regions. This improves accessibility and SEO.

Try it

          

            
            
          

<header>
  <h1>My Site</h1>
  <nav><a href="#">Home</a> · <a href="#">About</a></nav>
</header>
<main>
  <article>
    <h2>Post Title</h2>
    <p>Content with <section> blocks.</p>
  </article>
  <aside>Sidebar</aside>
</main>
<footer>Copyright © <time datetime="2025">2025</time></footer>

Best practices

  • One h1 per page; use h2–h6 for subsections.
  • Label regions with semantic tags before divs.
  • Use alt text, title, lang, and metadata.
